# Cross-Atlas Reproducibility of GWAS-Driven Cell-Type Prioritization in Ulcerative Colitis

Semester research project, Muskaan, UC Santa Cruz, 2026.

## Overview

This project tests whether GWAS-driven cell-type prioritization
results replicate when run on independent scRNA-seq atlases
of the same tissue. Ulcerative colitis is the case study:
we run scDRS, scPagwas, and seismic on three independent UC
colon atlases (Smillie 2019, Kong 2023, Mennillo 2024) against
the de Lange 2017 UC GWAS, and quantify cross-atlas concordance
of cell-type rankings.
